*A estrutura sequencial refere-se a uma série de exercícios resolvidos conforme a ordem em que os comandos são executados no código. Não há uso de modularização, estruturas condicionais ou loops, justamente para que o aluno inicie o aprendizado em lógica de programação de forma mais direta e simplificada.*

---
